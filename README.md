# Udacity Blockchain Capstone

The capstone will build upon the knowledge you have gained in the course in order to build a decentralized housing product. 

## Install Packages

Move to contracts directory, install dependencies and compile contracts:

```
cd eth-contracts
npm install
truffle compile
```

## Run Tests

Start ganache-cli in a seperate terminal within the eth-contracts directory. Migrate the contracts and run the test files.

```
ganache-cli -m "faculty gloom health bachelor another goose region patch silver reduce rifle erase"
truffle migrate --reset
truffle test ./test/TestERC721Mintable.js
truffle test ./test/TestSquareVerifier.js
truffle test ./test/TestSolnSquareVerifier.js
```

## Setup ZoKrates

Install a ZoKrates zkSnarks dev environment using Docker.

Instructions to installing docker [here](https://docs.docker.com/install/).

```
docker run -v <path to your project folder>:/home/zokrates/code -ti zokrates/zokrates /bin/bash
cd code/square
~/zokrates compile -i square.code
~/zokrates setup
~/zokrates compute-witness -a 3 9
~/zokrates generate-proof
~/zokrates export-verifier
```

## Rinkeby Deployment

Run the following command to deploy the contracts onto the Rinkeby Network:

```
truffle migrate --reset --network rinkeby
```

## Mint Tokens

Use [Remix](https://remix.ethereum.org/) or [MyEtherWallet](https://www.myetherwallet.com/access-my-wallet) to mint tokens to see in Opensea. Use SolnSquareVerifier's abi and contract address.
